DIRECTORIO EMPRESARIAL DE SANTANDER

Política de Cookies y Almacenamiento Local

LSSI-CE art. 22.2 · RGPD (UE) 2016/679

Volver al directorio

El Directorio de Áreas Empresariales e Industriales de Santander está desarrollado por el Ayuntamiento de Santander con especial atención a la privacidad de sus usuarios. La presente política describe con precisión qué tecnologías de almacenamiento utiliza este sitio web, con qué finalidad y cómo puede gestionarlas.

Sobre las cookies tradicionales

Este sitio web no utiliza cookies propias de seguimiento, publicitarias ni de perfilado. No se instalan cookies en su navegador por parte del servidor de este sitio.

Las cookies son pequeños ficheros de texto que los servidores web pueden almacenar en el navegador del usuario. A diferencia de otros sitios web, este directorio ha sido diseñado para no depender de cookies para ninguna de sus funcionalidades principales.

Almacenamiento local (localStorage)

Este sitio web utiliza el almacenamiento local del navegador (localStorage), una tecnología estándar del navegador que funciona de forma similar a las cookies pero con diferencias importantes:

  • Los datos se almacenan exclusivamente en su dispositivo y no se envían automáticamente al servidor con cada petición.
  • La información permanece en su navegador hasta que usted la elimine o expire el tiempo de retención configurado.
  • A diferencia de las cookies, el almacenamiento local no tiene fecha de caducidad automática.
Detalle de los elementos almacenados
Clave (key) Tipo Finalidad Datos almacenados Duración
analytics_log Analítica Registro local de interacciones para análisis estadístico del uso del directorio (empresas consultadas, búsquedas realizadas, filtros usados). Array de hasta 10.000 eventos: tipo de acción, identificador de empresa, marca de tiempo, sector. Hasta que se limpie el almacenamiento del navegador. Máximo 12 meses.
analytics_consent Funcional Almacena la decisión del usuario sobre el consentimiento para el uso de analítica local, evitando mostrar el banner de consentimiento en cada visita. Cadena de texto: "accepted" o "rejected". Permanente hasta que se limpie el almacenamiento.
solicitudes_cambios Copia seguridad Copia de seguridad local de los formularios de solicitud de modificación de datos enviados, para referencia del usuario en caso de necesidad. Array de objetos con: nombre, cargo, teléfono, empresa concernida, tipo de solicitud, fecha de envío. Sin contraseñas ni datos bancarios. Permanente hasta que se limpie el almacenamiento.
recent_companies Funcional Lista de las últimas empresas consultadas en el directorio para facilitar el acceso rápido a fichas visitadas recientemente. Array de hasta 8 identificadores numéricos de empresas. Permanente hasta que se limpie el almacenamiento.
dash_auth Funcional Almacena el estado de autenticación de acceso al panel de gestión del directorio (uso exclusivo de los gestores del Ayuntamiento). Token de sesión de gestión. No contiene datos personales de usuarios públicos. Sesión del navegador o hasta cierre de sesión.
tutorial_seen Funcional Registra si el usuario ya ha visualizado el tutorial de uso del mapa interactivo, para no mostrarlo en visitas posteriores. Cadena de texto: "1". Permanente hasta que se limpie el almacenamiento.
Analítica web: Plausible Analytics

Este sitio web utiliza Plausible Analytics, una herramienta de análisis web respetuosa con la privacidad que ha sido seleccionada específicamente por no utilizar cookies ni recopilar datos personales identificables.

  • Plausible no instala ninguna cookie en su navegador.
  • Plausible no recopila ni almacena datos personales: no registra su dirección IP completa, no utiliza identificadores de usuario ni fingerprinting del dispositivo.
  • Los datos recopilados son estadísticas anonimizadas: número de visitas, páginas consultadas, país de origen (a nivel de país, sin precisión geográfica), tipo de dispositivo y navegador.
  • No está sujeto a las obligaciones de consentimiento del artículo 22.2 LSSI-CE ni del RGPD para herramientas de analítica sin cookies.
  • Más información en la política de privacidad de Plausible: plausible.io/privacy
Recursos de terceros

El sitio web carga recursos de los siguientes servicios de terceros, que pueden establecer sus propias cookies o registrar datos técnicos de acceso conforme a sus respectivas políticas:

Google Fonts
fonts.googleapis.com · fonts.gstatic.com
Se carga la tipografía Inter para la interfaz del directorio. Google puede registrar datos técnicos de conexión (dirección IP, agente de usuario) conforme a su política de privacidad. Política: policies.google.com/privacy
MapLibre GL / OpenStreetMap
unpkg.com · openstreetmap.org tiles
Librería de mapas de código abierto para el mapa interactivo del directorio. Las teselas del mapa se cargan desde servidores de OpenStreetMap, que pueden registrar accesos técnicos. Política: osmfoundation.org/wiki/Privacy_Policy
Tailwind CSS CDN
cdn.tailwindcss.com
Marco de estilos CSS cargado desde la CDN oficial de Tailwind. Puede registrar datos técnicos de conexión. Se utiliza únicamente para el estilo visual de la interfaz.
Iframes de Street View / Panoramas 360°
Contenido embebido bajo demanda del usuario
Cuando el usuario accede a la vista panorámica de una empresa, se carga un iframe externo con el Street View o panorama correspondiente. Estos recursos solo se cargan cuando el usuario hace clic explícitamente en "Ver Street View". El servicio de Street View puede establecer sus propias cookies. Política de Google: policies.google.com/privacy
Cómo gestionar y eliminar el almacenamiento local

Puede eliminar todos los datos almacenados localmente por este sitio web siguiendo las instrucciones de su navegador:

Google Chrome
Configuración (⋮) → Más herramientas → Herramientas para desarrolladores → Application → Local Storage → Seleccionar el dominio → Clic derecho → Clear. O bien: Configuración → Privacidad y seguridad → Borrar datos de navegación → Cookies y otros datos de sitios.
Mozilla Firefox
Herramientas → Información del sitio web → Almacenamiento → Almacenamiento local. O bien: Configuración → Privacidad y seguridad → Cookies y datos del sitio → Borrar datos.
Microsoft Edge
Configuración (···) → Configuración → Privacidad, búsqueda y servicios → Borrar datos de exploración → Cookies y otros datos de sitios.
Safari
Preferencias → Privacidad → Administrar datos de sitios web → Eliminar. O bien: Preferencias → Avanzado → Mostrar menú Desarrollo → Desarrollo → Vaciar cachés.

Eliminar el almacenamiento local de este sitio no afectará a su navegación general. Únicamente perderá el historial de empresas visitadas recientemente, la preferencia de consentimiento (se mostrará de nuevo el banner) y el registro de analítica local.

Consentimiento y retirada del consentimiento

Cuando visite por primera vez este sitio web, se le mostrará un banner informándole sobre el uso del almacenamiento local para analítica. Puede aceptar o rechazar dicho uso:

  • Si acepta: Se activará el registro de analítica local (analytics_log). Siempre podrá cambiar su decisión eliminando la clave analytics_consent del almacenamiento local de su navegador.
  • Si rechaza: No se almacenará ningún registro de analítica. Las funcionalidades básicas del directorio (mapa, búsqueda, fichas de empresa) seguirán funcionando con total normalidad.

El consentimiento para el almacenamiento funcional (empresas recientes, preferencias de interfaz) es necesario para el correcto funcionamiento de estas características. Puede suprimirlo en cualquier momento eliminando los datos de almacenamiento local de su navegador según las instrucciones anteriores.

Actualizaciones de esta política

El Ayuntamiento de Santander puede actualizar la presente Política de Cookies y Almacenamiento Local cuando se incorporen nuevas funcionalidades al sitio web o cuando cambien los servicios de terceros utilizados. Le recomendamos revisar periódicamente esta página.

Para cualquier consulta sobre esta política, puede contactar con el Delegado de Protección de Datos en dpd@santander.es.
